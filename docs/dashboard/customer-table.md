# Customer Data Table

## Purpose

The main content area displays a sortable data table with detailed customer information. This table serves as the primary reference for customer data across the platform.

## Table Columns

| Column | Description |
|--------|-------------|
| Customer / Firm Name | Business name |
| Customer Name | Contact person name |
| District | Customer's district |
| State | Customer's state |
| Customer Code | Unique customer identifier |
| Phone | Contact phone number |
| GSTIN | GST identification number |
| Total Sales | Aggregated sales amount |
| Category Name/Code | Customer category |
| Average Sales | Average sales per period |
| Total Invoices | Invoice count |
| Total Cgst | Total central GST |

> 💡 **Tip:** Click column headers to sort. Use the settings overlay to toggle visible columns.

## Sorting

Click any column header to sort the table in ascending or descending order. The active sort column is indicated by an arrow icon.

## Configuration

You can customize which columns appear in the table via the **Sales View Settings** overlay. See [Sales View Settings](../settings/sales-view-settings) for details.
