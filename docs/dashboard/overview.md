# Dashboard Overview

## Purpose

The Home Dashboard serves as the landing page after login. It provides a high-level overview of key business metrics and quick access to all ZoTok modules.

## When to Use

Use the Dashboard as your starting point each day to review:

- Outstanding receivables
- Sales performance month-to-date
- Customer data and activity

## Key Elements

The Dashboard is divided into three main sections:

1. **Navigation Sidebar** — Access all platform modules
2. **KPI Cards** — Key performance indicators for the current month
3. **Customer Data Table** — Sortable table with customer details

![Dashboard Layout](/screenshots/dashboard/01_Dashboard_Home_enhanced.jpg)

## Navigation Sidebar

The left sidebar contains navigation items for all platform modules:

| Module | Icon | Description |
|--------|------|------------|
| **Home** | 🏠 | Dashboard overview with KPI cards and customer data |
| **Sales** | 💰 | Sales transactions, charts, column configuration |
| **Payments** | 💳 | Payment tracking, age-wise receivables |
| **Item** | 📦 | Product catalog, inventory tracking |
| **Dashboard** | 📊 | Advanced analytics and charts |
| **Day Book** | 📖 | Daily transaction register |
| **Threads** | 💬 | Customer conversations, queries, and signal inbox |
| **Grow** | 🚀 | Campaign management and customer lifecycle |

> 💡 **Tip:** The Home Dashboard is your command center — all modules are accessible from the left sidebar.
