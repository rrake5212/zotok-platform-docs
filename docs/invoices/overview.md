# Invoices Module

## Overview

The Invoices module manages the generation and tracking of invoices. It is accessible from the main navigation sidebar.

:::info
This module is visible in the platform sidebar. A dedicated walkthrough video is not yet available. This page will be updated with detailed screenshots and step-by-step instructions when the walkthrough is processed.
:::

## Accessing Invoices

Click **Invoices** (🧾 icon) in the left navigation sidebar.

## Expected Features

Based on the platform sidebar and related modules, the Invoices module is expected to include:

- Invoice list with search and filters
- Create new invoices
- Invoice generation from orders
- Invoice status tracking (paid, unpaid, overdue)
- GST invoice management
- Invoice printing and download
