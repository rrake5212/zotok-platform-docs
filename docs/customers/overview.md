# Customers Module

## Overview

The Customers module provides a centralized view of all customer data. It is accessible from the main navigation sidebar.

:::info
This module is visible in the platform sidebar. A dedicated walkthrough video is not yet available. This page will be updated with detailed screenshots and step-by-step instructions when the walkthrough is processed.
:::

## Accessing Customers

Click **Customers** (👥 icon) in the left navigation sidebar.

## Expected Features

Based on the platform sidebar and related modules, the Customers module is expected to include:

- Customer list with search and filters
- Customer profile and details
- Customer categorization (by type, district, state)
- Contact information management
- Customer code and GSTIN lookup
