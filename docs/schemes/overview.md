# Schemes Module

## Overview

The Schemes module manages discount and promotional schemes. It is accessible from the main navigation sidebar.

:::info
This module is visible in the platform sidebar. A dedicated walkthrough video is not yet available. This page will be updated with detailed screenshots and step-by-step instructions when the walkthrough is processed.
:::

## Accessing Schemes

Click **Schemes** (🏷️ icon) in the left navigation sidebar.

## Expected Features

Based on the platform sidebar, the Schemes module is expected to include:

- Scheme creation and management
- Discount configuration
- Scheme applicability rules
- Scheme calendar and scheduling
- Scheme performance tracking
