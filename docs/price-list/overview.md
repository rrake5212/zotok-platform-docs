# Price List Module

## Overview

The Price List module manages product pricing and price lists. It is accessible from the main navigation sidebar.

:::info
This module is visible in the platform sidebar. A dedicated walkthrough video is not yet available. This page will be updated with detailed screenshots and step-by-step instructions when the walkthrough is processed.
:::

## Accessing Price List

Click **Price List** (💲 icon) in the left navigation sidebar.

## Expected Features

Based on the platform sidebar, the Price List module is expected to include:

- Product price list management
- Price list creation and versioning
- Customer-specific pricing
- Bulk price updates
- Price list export
