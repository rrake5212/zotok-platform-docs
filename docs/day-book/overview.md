# Day Book Module

## Overview

The Day Book module provides a daily register of all financial transactions. It is accessible from the main navigation sidebar.

:::info
This module is visible in the platform sidebar. A dedicated walkthrough video is not yet available. This page will be updated with detailed screenshots and step-by-step instructions when the walkthrough is processed.
:::

## Accessing Day Book

Click **Day Book** (📖 icon) in the left navigation sidebar.

## Expected Features

Based on the platform sidebar, the Day Book module is expected to include:

- Daily transaction log
- Transaction type filtering (sales, payments, receipts)
- Date-wise transaction view
- Opening and closing balances per day
- Transaction search and export
