# Campaigns (Grow) Overview

## Overview

The Grow module powers ZoTok's campaign management capabilities. It enables businesses to create targeted marketing campaigns, manage customer lifecycles, and engage customers through automated messaging.

## Accessing Campaigns

Click **Grow** (🚀 icon) in the left navigation sidebar.

## Module Sections

The Grow module has two main sections:

1. **Campaigns** — Create and manage marketing campaigns
2. **Customer Lifecycle** — Manage customer groups and lifecycle workflows

![Grow Sidebar](/screenshots/campaigns/grow-sidebar.jpg)

## Sidebar Navigation

When you enter Grow, the sidebar shows:

| Item | Description |
|------|-------------|
| **Campaign** | Campaign creation and management |
| **Customers** | Customer group management |

## Customer Group Management

The Grow module organizes customers into groups based on their engagement status:

| Group | Description |
|-------|-------------|
| **Active** | Currently engaged customers (e.g., 20 customers) |
| **Activated** | Recently onboarded customers |
| **Dormant** | Inactive customers needing re-engagement |
| **High Invoices** | Customers with high invoice volumes |
| **Irrelevant** | Non-target customers |
| **Low Payment** | Customers with low payment activity |
| **New Lead** | Prospective customers (e.g., 525 customers) |
| **To Be Inactive** | Customers nearing deactivation |
| **Warm** | Engaged but not yet converted |
