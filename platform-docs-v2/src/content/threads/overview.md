# Threads Module

## Overview

The Threads module is ZoTok's conversation management system. It organizes all customer interactions — queries, follow-ups, issues, and signals — in a unified inbox.

## Accessing Threads

Click **Threads** (💬 icon) in the left navigation sidebar.

## Threads Interface

![Threads Dashboard](/screenshots/threads/03_Threads_Dashboard_enhanced.jpg)

Upon navigating to Threads, the interface displays:

- **Header** — Shows "Threads" with customer count (e.g., 8 Customers)

### Tabbed Navigation

| Tab | Purpose |
|-----|---------|
| **Groups** | Group conversations (broadcast to multiple customers) |
| **Threads** | Individual conversation threads with customers |
| **Issues** | Escalated tickets and problem reports |
| **Signal Inbox** | Incoming automated signals and alerts |

### Category Filters

- **All Categories** — Show all conversation types
- **Unactioned** — Show only conversations requiring action

## Thread Types

Common conversation categories include:

| Thread Type | Description |
|-------------|-------------|
| New Query Flow | First-time customer inquiries |
| Bulk Order Negotiation | Large order discussions |
| Payment Follow-up | Outstanding payment reminders |
| Product Catalogue / Browsing Request | Catalog sharing |
| Company Info / About Request | Business information queries |
| Conversation Initiation | Simple greeting / onboarding |

## Empty Conversation State

![Thread Detail View](/screenshots/threads/04_Thread_Detail_View_enhanced.jpg)

When no conversation is selected, the system displays:

> "Select a conversation from the list to view messages"

> 💡 **Tip:** Conversations are automatically categorized. Ensure query types are configured for accurate routing.
