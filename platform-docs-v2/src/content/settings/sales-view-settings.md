# Sales View Settings

## Overview

The Sales View Settings overlay allows you to customize how sales data is displayed on your dashboard.

## Accessing Settings

Click **Sales** (💰 icon) in the left navigation sidebar to open the settings overlay.

## Configuration Options

### Chart Configuration

- **Chart 1 & 2 Selection** — Choose which charts to display on the dashboard
- **Donut Chart Toggle** — Enable or disable the donut chart visualization

### Table Columns

Customize which columns appear in the customer data table:

| Column | Description |
|--------|-------------|
| customerFirmName | Customer business name |
| customerName | Contact person name |
| District | Customer district |
| State | Customer state |
| customerCode | Unique customer code |
| phone | Contact phone number |
| gstin | GST identification number |
| Total Sales | Aggregated sales amount |
| Category | Customer category name/code |
| Average Sales | Average sales per period |
| Total Invoices | Invoice count |
| Total Cgst | Total central GST |

### Data Filters

- **Type** — Filter by transaction type
- **Customer** — Filter by specific customer
- **Category** — Filter by customer category

### State-wise Summary

A state-wise summary table shows aggregated sales data by state.

## Actions

- **Save** — Apply your configuration
- **Cancel** — Discard changes and return to dashboard
