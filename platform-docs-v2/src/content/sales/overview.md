# Sales Module

## Overview

The Sales module provides tools for tracking sales transactions, viewing sales charts, and configuring which data columns to display.

## Accessing Sales

Click **Sales** (💰 icon) in the left navigation sidebar. This opens the **Sales View Settings** overlay where you can configure your sales view.

## Sales View Settings

![Sidebar Menu](/screenshots/dashboard/02_Sidebar_Menu_Open_enhanced.jpg)

The Sales overlay provides:

### Chart Configuration

- **Chart 1 & 2 Selection** — Choose which charts to display
- **Donut Chart Toggle** — Enable/disable donut chart view

### Table Columns

Check or uncheck columns to customize the customer data table:

- customerFirmName, customerName, District, State, customerCode
- phone, gstin, Total Sales, Category, Average Sales
- Total Invoices, Total Cgst

### Data Filters

- **Type** — Filter by transaction type
- **Customer** — Filter by specific customer
- **Category** — Filter by customer category

### State-wise Summary

A state-wise summary table shows aggregated sales (e.g., Telangana: ₹900+).

## Actions

- **Save** — Apply your configuration changes
- **Cancel** — Discard changes

> 💡 **Tip:** Configure your most-used columns first so the table shows only what you need.
