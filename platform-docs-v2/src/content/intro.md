# ZoTok Platform Manual

**Version 1.0 · July 2026**

Welcome to the ZoTok Platform User Manual. ZoTok is a comprehensive field operations and distributor management platform designed to streamline communication, sales tracking, inventory management, and payment reconciliation for businesses.

## Who Is This Manual For?

- **New Joiners** — Learn the platform end-to-end
- **Sales Team** — Complete sales workflows
- **Account Team** — Handle financial operations
- **Customer Success Team** — Resolve customer queries
- **Administrators** — Configure and manage the platform

## Platform Overview

The left sidebar provides access to all core modules:

| Module | Description |
|--------|-------------|
| **Home** | Dashboard with KPI cards, receivables ageing, customer data |
| **Sales** | Sales transactions, charts, column configuration |
| **Payments** | Payment tracking, age-wise receivables |
| **Item** | Product catalog, inventory management |
| **Dashboard** | Advanced analytics and charts |
| **Day Book** | Daily transaction register |
| **Threads** | Customer conversations and query management |
| **Grow** | Campaign management and customer lifecycle |

Start with the [Login](getting-started/login.md) guide, then explore the [Dashboard](dashboard/overview.md) to understand your business at a glance.
