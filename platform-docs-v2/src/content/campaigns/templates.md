# Campaign Templates

## Overview

Campaign templates are pre-built message sequences that can be used to quickly create campaigns. ZoTok provides both Default and Custom templates.

## Template Library

![Template Library](/screenshots/campaigns/template-library.jpg)

The Template Library is organized into two tabs:

- **Default** — Pre-built templates provided by ZoTok
- **Custom** — Templates created by your organization

## Available Templates

### Default Templates

![Template Preview](/screenshots/campaigns/template-preview.jpg)

| Template | Description |
|----------|-------------|
| **Simple Greeting** | A bot that greets users with a friendly welcome message |
| **Basic Introduction** | Introduces your business, product, or service in a simple and structured way |
| **Basic Survey Form** | Collects basic feedback or responses from users through a short survey form |
| **Lead Generator** | Captures potential customer information (name, email, phone) for follow-up |
| **Google Sheet WhatsApp Campaign Bot** | Automatically sends personalized WhatsApp messages to customers at scheduled times |
| **New Agent** | Template for introducing a new agent |
| **New Bot** | Template for introducing a new bot |
| **Trym** | Trym-specific template |

![Template Types](/screenshots/campaigns/template-types.jpg)

## Using a Template

1. Navigate to the **Template Library** during campaign creation
2. Browse available templates
3. Click **Preview** to see the template content
4. Click **Use This Template** to apply it to your campaign

## Tips

- Preview templates before using to ensure they match your campaign goals
- Create custom templates for recurring campaign types
- Combine multiple templates for multi-step journey campaigns
