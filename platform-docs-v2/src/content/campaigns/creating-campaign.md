# Creating Campaigns

## Overview

The Campaigns section allows you to create targeted marketing campaigns. ZoTok supports **Journey Campaigns** — automated multi-step campaigns that guide customers through a predefined sequence.

## Campaigns Dashboard

![Campaigns Empty State](/screenshots/campaigns/campaigns-empty.jpg)

The Campaigns dashboard shows:

- List of existing campaigns (filtered by "Last 30 days" by default)
- **Create Campaign** button to start a new campaign
- Search and pagination controls

## Creating a Journey Campaign

![Create Journey Campaign](/screenshots/campaigns/create-journey-campaign.jpg)

1. Click **Create Campaign**
2. Select **Journey Campaign** type
3. Fill in the basic details:

| Field | Required | Description |
|-------|----------|-------------|
| Campaign Name * | Yes | A unique name (max 60 characters) |
| Description * | Yes | Campaign purpose (min 50 characters, max 300) |

4. Click **Create Campaign**

## Campaign Setup Steps

A campaign is configured through 5 steps:

![Create Campaign Form](/screenshots/campaigns/create-campaign.jpg)

### 1. Templates

Select the messaging template for each step of your campaign journey. You can choose from:
- **Default Templates** — Pre-built templates
- **Custom Templates** — Your organization's custom templates

### 2. Details

Configure campaign details:

- **Campaign Name** — A descriptive name
- **Header** — Message header text
- **Image** — Upload campaign image (PNG, recommended 1280×720px)

### 3. Products

Select products to feature in the campaign.

### 4. Audience

Define the target audience by selecting customer groups and applying filters.

### 5. Settings

Configure campaign schedule and delivery settings.

## Tips

- Use descriptive campaign names for easy identification
- Upload high-quality images (1280×720px recommended)
- Define clear audience segments for better targeting
- Preview your campaign before launching
